
1 -> git clone <repo>

2 -> cd <proj_dir>

3 -> npm install
    
<<-------   Build project  --------->>

4 -> npm run build

'dist' folder will be generated in root-directory 

<<---------- Testing project -------->>
    
    ------ Classic web

           cd <proj_dir>
           cd examples/classic_web
           run some server to host index.html
    
    ------ ReactJs

           cd <proj_dir>
           cd examples/reactjs
           npm install
           npm start
    
    ------ NextJs

           cd <proj_dir>
           cd examples/nextjs
           npm install 
           npm run dev
